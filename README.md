# ScaleX Landing Page

A responsive landing page inspired by the ScaleX website, built using HTML and CSS.

## 🚀 Features

- Responsive design
- Modern UI
- Clean layout
- CSS positioning
- Mobile-friendly
- Smooth hover effects

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

├── index.html
├── style.css
└── ScaleX-logo.png

## 🎯 Purpose

This project was created as part of a frontend web development assignment to practice responsive layouts and CSS positioning.
